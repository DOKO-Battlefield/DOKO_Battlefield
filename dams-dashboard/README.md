# D'AMS Dashboard

The D'AMS Dashboard provides a centralized interface for monitoring wearable telemetry, active sessions, and athlete data in real time.

It is designed for coaches, event staff, and administrators to visualize data collected by the D'AMS Service.

---

# Purpose

The dashboard transforms raw telemetry into an easy-to-understand interface, allowing staff to monitor connected wearable devices and athlete activity during training sessions and live events.

---

# Current Prototype (V1)

The first version demonstrates real-time communication with the D'AMS Service and displays incoming telemetry from connected wearable devices.

Current capabilities include:

- Live telemetry monitoring
- Session visualization
- Device status
- Athlete data display
