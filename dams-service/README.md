# D'AMS V1

Adaptive Motion + Safety Intelligence System

## Purpose

D'AMS monitors athlete movement and strain in real time.

The system evaluates:

- Motion intensity
- Heart rate
- Fatigue
- Safety status

## States

NORMAL
SLOW
STOP

## Hardware

- ESP32
- MPU6050
- Pulse Sensor
- RGB LED
- Buzzer

## Software

Backend:
- Flask
- SQLAlchemy
- SQLite

Frontend:
- React
- Vite

## Features

- Real-time telemetry
- Session tracking
- Alert system
- Historical analytics
- Dashboard monitoring
