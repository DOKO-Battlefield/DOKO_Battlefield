# DOKO Client

The DOKO Client is the main public-facing website for DOKO Battlefield. It presents the brand, experiences, booking options, media, investor information, and future platform concepts.

---

## Purpose

This client is designed to introduce users to the DOKO ecosystem and support the public website experience.

It includes two major directions:

1. **Current public website**
   - DOKO brand
   - Experiences
   - Booking
   - Restaurant/menu content
   - Media vault
   - FAQ
   - Contact
   - Investor information

2. **Future platform concepts**
   - NAQI teaser
   - DOKO world-building
   - Expanded media and immersive experience pages
